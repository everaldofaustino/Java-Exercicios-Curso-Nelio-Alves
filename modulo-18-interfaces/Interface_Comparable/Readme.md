




# 📊 Employee Manager - Ordenação de Funcionários

## 📋 Descrição do Projeto

Este é um programa simples em Java que lê um arquivo CSV contendo dados de funcionários (nome e salário), armazena os registros em uma lista, ordena os funcionários por nome e exibe o resultado ordenado no console.

## 🎯 Funcionalidades

- ✅ Leitura de arquivo CSV com dados de funcionários
- ✅ Armazenamento dos dados em uma lista (`ArrayList`)
- ✅ Ordenação alfabética dos funcionários por nome
- ✅ Exibição formatada dos resultados no terminal
- ✅ Tratamento básico de erros (arquivo não encontrado, formato inválido)
