## 📚 Sobre o Curso

Curso abrangente que cobre desde os fundamentos da linguagem Java até tópicos avançados como:

- Programação Orientada a Objetos (POO)
- Tratamento de exceções
- JDBC com PostgreSQL
- JPA / Hibernate
- Spring Boot
- Web services com REST API
